# Sharp for AWS Lambda

Generate a Node.js Lambda Layer with [Sharp](https://github.com/lovell/sharp)

| Sharp | Built for Node.js |
| :-- | :-- |
| Latest | LTS |

### Build

```bash
# download the code
$ git clone \
    --depth 1 \
    https://github.com/jeromedecoster/sharp-lambda-layer.git \
    /tmp/aws

# cd
$ cd /tmp/aws

# setup and build
$ make setup
$ make build
```
